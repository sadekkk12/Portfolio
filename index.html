Current Project - BrainDeck

Overview

BrainDeck is an IOS app that allows you to create customized flashcards and put them in a bundle/category of your choosing. You can then use these bundles to run a Study Mode, which lets you practice these flashcards and also informs you which cards need more practice.

Progress and Learning

I am in the earlier stages of the development. I have created the foundational parts of my Model, while implementing the MVC pattern. I have my first take on some of the UI and am now focusing on making it possible to create categories/flashcards through the UI. Followed by implementing data persistence through CoreData to store the categories and flashcards locally inside the device.
