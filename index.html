Current Project - BrainDeck

Overview

BrainDeck is an IOS app that allows you to create customized flashcards and put them in a bundle/category of your choosing. You can then use these bundles to run a Study Mode, which lets you practice these flashcards and provides feedback.

Progress and Learning

I am in the earlier stages of the development. I have created the foundational parts of my Model, while implementing the MVC pattern. I have my first take on some of the UI and currently making it possible to create flashcards through the UI. Followed by implementing data persistence through CoreData to store the categories and flashcards locally inside the device.
